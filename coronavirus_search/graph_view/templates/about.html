Les données utilisées sur ce site proviennent de Google trend et de Coronavirus COVID19 API. Google Trend renvoit des données normalisées de recherche sur le moteur Google. Ces données sont normalisées pour une période donnée et une région du monde. Coronavirus COVID19 API renvoit, entre autres, les cas de COVID19 pour une période donnée et une région du monde.

Ce projet a été principalement écrit en Python. La librairie pytrends a été utilisée pour accéder aux données de Google Trend. Le site a été construit en utilisant le framework Django. Les graphiques ont été tracés avec la librairie Plotly
